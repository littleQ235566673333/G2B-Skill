To compute the weighted average denomination, sum the product of the number of each bill type and its denomination, then divide by the total number of bills.

We have values (from June 30, 2011) for each denomination's total value in circulation:

| Denomination | Value ($)            |
|--------------|----------------------|
| $1           | 9,813,612,968        |
| $2           | 1,834,819,666        |
| $5           | 11,348,207,260       |
| $10          | 16,444,843,570       |
| $20          | 134,976,537,960      |
| $50          | 67,796,754,100       |
| $100         | 745,181,164,800      |
| $500         | 142,177,500          |
| $1,000       | 165,495,000          |
| $5,000       | 1,765,000            |
| $10,000      | 3,450,000            |
| Fractional   | 600                  |

We solve for the count of each bill type by dividing each denomination value by its denomination. The weighted average = total value / total count.
